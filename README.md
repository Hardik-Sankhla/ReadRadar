# ReadRadar

ReadRadar is the frontend viewer for an **Autonomous Knowledge Operating System**. It is an intelligence dashboard designed for AI engineers and researchers to discover, validate, and navigate technical knowledge assets—ranging from books and research papers to GitHub repositories and courses.

Built by [Hardik Sankhla](https://github.com/Hardik-Sankhla), ReadRadar is not just a static directory. It is the presentation layer for a living, breathing knowledge graph maintained entirely by autonomous AI agents.

## 🚀 The Vision

Traditional book directories answer the question: *"What books exist?"*

ReadRadar answers the question: *"What should I learn next, why, and how trustworthy is this resource?"*

By representing each item as a generalized **Knowledge Asset**, ReadRadar provides:
- **Knowledge DNA**: An at-a-glance visualization of the asset's conceptual depth (Math, Code, Theory, Production).
- **Prerequisite Intelligence**: Clear dependencies showing what you need to know before reading.
- **Trust Indicators**: Verified confidence scores backed by community mentions, official sources, and agent validation.
- **Learning ROI**: Estimated time requirements and explicit learning outcomes.

## 🏗️ Architecture

ReadRadar is strictly a **Viewer** repository. It relies on a decoupled, modular architecture:

1. **KnowledgeOS** (The Workers): A suite of autonomous agents that discover, validate, and enrich metadata across the internet (GitHub, ArXiv, Publishers, Telegram).
2. **KnowledgeRegistry** (The Truth): A standalone repository that acts as a Git-backed NoSQL database, storing the rich JSON and Markdown structures generated by KnowledgeOS.
3. **ReadRadar** (The Viewer): This repository. It consumes `KnowledgeRegistry` and renders them into the neo-brutalist intelligence dashboard you see. ReadRadar is completely ignorant of how the data is generated, making the frontend lightning fast, scalable, and independent.

## 🛠️ Tech Stack
- **Framework**: Astro (Static Site Generation)
- **UI Components**: React & Tailwind CSS v4
- **Design Language**: Neo-brutalist Dark Mode
